BetterMB

A CLI client for the SU meal booking system that offers a better user experience than the one provided by Stellenbosch University. The name "BetterMB" is still up for debate.

For now, BetterMB is mainly useful for those who know the basics of the command line. A GUI version will be released eventually.

Requirements

  • Java 21

Install

  1. Download the archive for you operating system
  2. Extract the archive
  3. cd extracted_path/bin
  4. ./bettermb
  5. (Optional) Add bettermb to path
