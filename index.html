BetterMB

Classic View Normal View

A program to substitute the SU meal booking system that offers a better user experience than the very low bar set by Stellenbosch University.
The name "BetterMB" is still up for debate.

Requirements

  • Java 21
  • Google Chrome

Install and run

  1. Download and extract the zip file
  2. Open the extracted folder
  3. Open bin
  4. For Windows: run bettermb-<type>-<version>.bat
    For Linux: run bettermb-<type>-<version>
  5. Create a shortcut so it is easy to find again
